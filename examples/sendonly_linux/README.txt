使用方法
------------------------------------------------------------------
1. 先编译
mkdir build
cd build
cmake ..
make

------------------------------------------------------------------
2. 执行
./rdlc_demo /dev/ttyUSB0 115200

-----------------------------------------------------------------
3. 现象
Linux设备将会通过ttyUSB0发送{0x11,0x22,0x33}到地址{0x01,0x02}，频率10ms一次。
